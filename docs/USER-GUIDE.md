# 手把手改造 TRAE Work 主题

这篇教程不要求你懂代码。准备一台 Mac、官方 TRAE Work 和一张喜欢的图片就够了。

## 第一步：下载

在 GitHub 页面点击 **Code → Download ZIP**，下载后双击解压。

打开解压后的文件夹，再打开 `macos`。

## 第二步：安装

双击 `Install TRAE Work Dream Skin.command`。

如果系统不让打开：

1. 右键这个文件。
2. 选择“打开”。
3. 在弹窗里再次点“打开”。

脚本会检查你装的是不是官方 TRAE Work。应用已经打开时，先保存输入框里还没发送的内容，再按提示重启。

看到安装完成提示后，桌面会多出四个文件：启动、定制、验证和恢复。

## 第三步：换图片

双击桌面的 `Customize TRAE Work Dream Skin.command`。

选择图片时，尽量满足下面三点：

- 横图，宽度最好在 2000 像素以上；
- 主体放在中间偏右，左边留一些空白给任务列表；
- 文字区域不要太花，避免标题和输入框看不清。

然后输入主题名字。工具会压缩图片并刷新主题，不会改动原图。

## 第四步：检查

双击桌面的 `Verify TRAE Work Dream Skin.command`。

重点看四处：

1. 左侧任务列表能不能看清；
2. 中间输入框能不能点击和输入；
3. Work、Code、Design 等按钮是否正常；
4. 图片有没有挡住重要内容。

如果颜色不合适，再运行一次定制工具即可。想换布局风格，可参考 [主题模板](THEMES.md)。

## 第五步：恢复

不想继续使用时，双击 `Restore TRAE Work Dream Skin.command`。

它只恢复官方外观，不会删除任务、项目、聊天或你选过的图片。

## 常见问题

### 双击后闪退

先确认系统是 macOS，并已安装官方 TRAE Work。右键安装文件选择“打开”，通常可以看到更完整的提示。

### 主题没有出现

双击桌面的启动文件。如果 TRAE Work 原来已经开着，保存未发送内容后让它重启一次。

### 更新 TRAE Work 后失效

先运行验证工具。新版页面结构可能发生变化，请在 GitHub Issue 中附上 macOS、芯片架构、TRAE Work 版本和错误提示，不要上传账号数据或私人聊天截图。
